# Case-para-Vara-de-Pesca-Projeto-para-Impressora-3D
Case modular para vara de pesca impresso em 3D. Projetado originalmente em TPU para maior flexibilidade e resistência, possui comprimento padrão de aproximadamente 1,20 m e pode ser facilmente expandido com módulos adicionais.
Características
Design modular e expansível
Comprimento padrão de aproximadamente 1,20 m
Pode ser adaptado para diferentes tamanhos de vara
Alças laterais opcionais para facilitar o transporte
Compatível com diferentes materiais de impressão 3D
Material Utilizado

O protótipo foi desenvolvido e testado em TPU, o que deixa o case mais flexível e resistente a impactos. No entanto, as peças também podem ser impressas em outros materiais, conforme a necessidade de cada usuário.

Montagem

As alças laterais são opcionais. Caso não deseje utilizá-las, basta imprimir apenas as peças do corpo (Body).

Como o projeto foi pensado para TPU, não recomendo utilizar apenas cola instantânea (Super Bonder ou similares) para a união das peças. A melhor opção é utilizar um soprador térmico, fundindo o material entre as partes de forma semelhante a uma soldagem. Isso proporciona uma união mais resistente e durável.

Personalização

O projeto foi criado para ser simples de modificar. Algumas possibilidades incluem:

Aumentar o comprimento do case imprimindo mais módulos Body;
Remover ou adicionar alças laterais;
Utilizar diferentes materiais de impressão;
Adaptar as dimensões para diferentes tipos de vara de pesca.
Observações

Antes de transportar equipamentos de maior valor, recomenda-se realizar testes para verificar a resistência da montagem e do material utilizado. A durabilidade final dependerá das configurações de impressão, do material escolhido e da qualidade da montagem.
